# tclips

[https://tclips.netlify.com/](https://tclips.netlify.com/)

### Quick Start
Copy repo:
```bash
git clone https://gitlab.com/moondef/tclips.git && cd tclips
```

Install deps:
```bash
yarn
```

#### Scripts
|`yarn <script>`    |Описание|
|-------------------|-----------|
|`run`            |Run app on `localhost:1234`|
|`build`            |Build app to ./dist|
